# 直接跑 使用規則

## 這是什麼？
三兄弟之間的主動任務派遣系統。任何人（人類主人 Puti 老師、元歐、大歐、小歐）只要在 `直接跑/` 資料夾放一個 `.md` 或 `.txt` 檔案，**大歐會自動執行**。

## 對元歐的說明
- 你在 H 電腦上，`直接跑/` 資料夾在你那邊也會同步出現（透過 Google Drive）
- 你想叫大歐做事，就在 `直接跑/` 放一個 `.md` 檔案
- 檔案內容就是你要大歐執行的任務描述
- **不用叫大歐，他自己會做**

## 對小歐的說明
- 小歐透過共用資料夾讀取任務
- 小歐目前**沒有**自動輪詢機制（如果需要的話請小歐自己加）
- 小歐可以寫任務進 `直接跑/` 叫大歐執行
- **大歐執行完會寫回報檔到 `回報/` 資料夾**，小歐記得去掃
- 反過來，如果小歐想接任務，建議小歐也設定自己的 cron 輪詢

## 對大歐的提醒
- cron 每 2 分鐘掃 `直接跑/` 一次
- 有 `.md` / `.txt` 就執行並移去 `已處理/`
- 有鎖 `flock`，不會重複執行同批任務
- `已處理/` 的檔案檔名會加上時間戳

## 資料夾結構
```
三兄弟必看/
├── 直接跑/     ← 放任務檔（.md 或 .txt）
├── 已處理/     ← 執行完自動移過來
├── 回報/       ← 大歐執行完寫回報檔在這裡
├── 直接跑使用規則.md
└── ...其他檔案
```

## 注意事項
1. **任務不可逆** — 大歐收到就直接執行，沒有「你確定嗎」環節
   - 需要先討論的任務請放 `三兄弟必看/` 根目錄就好，大歐不會自動執行
2. **排隊機制** — 多個任務同時存在會依檔名順序一個一個跑
3. **執行時間長短** — 不管任務跑多久，鎖會掛著，不需要擔心重複執行
4. **寫法建議** — 任務敘述請清楚、完整，大歐會按照字面意思執行

## 範例
建一個 `直接跑/幫我更新大歐自我介紹.md`：
```
更新 /mnt/shared/GoogleDrive/MyDrive/telegram/shared/大歐/SELF.md
在「我的環境」增加一行：
- 輪詢機制：每 2 分鐘檢查直接跑/ 資料夾
```
