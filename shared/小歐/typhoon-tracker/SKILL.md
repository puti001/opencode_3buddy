# 🌀 颱風追蹤技能 (typhoon-tracker v2)

## 重要：資料來源

本技能**只參考以下 2 個中央氣象署網頁**，不再使用 Wikipedia 或其他來源：

### 來源一：颱風消息
```
https://www.cwa.gov.tw/V8/C/P/Typhoon/TY_NEWS.html
```
取得：目前所有颱風的編號、強度、位置、風速、氣壓、移動方向與速度。

### 來源二：屏東縣里港鄉鄉鎮預報
```
https://www.cwa.gov.tw/V8/C/W/Town/Town.html?TID=1001309
```
取得：目前天氣、溫度、體感溫度、降雨機率、相對溼度、舒適度。

> ⚠️ 這兩個網頁都是 JavaScript 動態渲染，**不能用 webfetch**。
> 必須用 **Playwright 瀏覽器工具**來取得頁面內容。

---

## 運作流程

### 1. 資料取得方式
使用 Playwright 瀏覽器依序：
1. `playwright_browser_navigate` → 颱風消息頁面
2. `playwright_browser_evaluate` → `() => document.body.innerText` 取文字
3. 解析文字中的颱風段落
4. `playwright_browser_navigate` → 鄉鎮預報頁面
5. `playwright_browser_evaluate` → `() => document.body.innerText` 取文字
6. 解析文字中的天氣表格與即時觀測

### 2. 解析颱風資料
從颱風頁面文字中，找「目前 (民國...) 太平洋地區有 X 個颱風」段落，
往下讀到每個颱風的名稱、編號、位置、風速、氣壓等資料。

需要的欄位：
- 颱風名稱、編號、強度分類
- 觀測時間
- 中心位置（緯度、經度）
- 移動方向與速度
- 中心氣壓
- 近中心最大風速（決定強度分級）
- 七級風暴風半徑

**強度分級對照**（CWA標準）：
- 輕度颱風：17.2~32.6 m/s
- 中度颱風：32.7~50.9 m/s
- 強烈颱風：51.0 m/s 以上

**位置換算人話**（預估，台灣中心約 23.5°N, 121°E）：
- 1°緯度 ≈ 110公里，1°經度 ≈ 92公里（台灣緯度）
- 台灣南端鵝鑾鼻約 21.9°N, 120.8°E
- 台灣北端富貴角約 25.3°N, 121.5°E
- 颱風在台灣哪個方向、距離多遠，用這些基準算

**風速比喻**（讓人聽得懂）：
- 17 m/s 以下：就風而已
- 17~25 m/s：騎車會被吹歪
- 25~33 m/s：撐傘會開花
- 33~41 m/s：瘦子站不穩
- 41~50 m/s：樹木斷枝、鐵皮飛
- 51 m/s 以上：房子會搖

### 3. 解析鄉鎮天氣資料
從鄉鎮頁面文字中找：
- **目前天氣概況**：「多雲午後短暫雷陣雨，天氣整體舒適，但仍有降雨機率」
- **即時觀測**：溫度 XX、體感溫度 YY、相對溼度 ZZ%
- **降雨機率列**：從表格找「降雨機率」那一列，讀出今日各時段的 %
- **溫度列**：從表格找「溫度」那一列，讀出今日最高最低
- **天氣狀況列**：從表格找「天氣狀況」那一列，看圖示文字描述

**三日重點**：從表格看今天、明天、後天的資料
- 今日（Day 1）：完整24小時逐時預報
- 明日（Day 2）：從00:00開始的資料
- 後日（Day 3）：從00:00開始的資料

### 4. 回報格式

**要講人話**，不要只是丟數字。像這樣：

```
🌀 米克拉快報
📍 在台灣鵝鑾鼻東南方約400公里海面上
🚶 龜速向北走，時速9公里（比腳踏車還慢）
💨 中颱等級，最大風速43m/s
   ——瘦子出門會被吹倒，鐵皮屋要注意
🌪️ 暴風圈半徑180公里，不算大隻但很紮實

🌤 里港鄉現在
🌡️ 28°C（體感34°C，悶熱）
💧 降雨：上午10%→午後60%，帶傘備用
   🕐 最可能下雨：下午4點前後

📅 未來3天里港天氣
   今天 27~34°C，午後雷陣雨
   明天 26~32°C，短暫陣雨
   後天 25~32°C，多雲午後陣雨
```

如果正在追蹤多個颱風就分別列出。如果資料抓不到就誠實說「目前氣象署網站資料還沒更新」。

## 狀態檔位置

```
/mnt/shared/GoogleDrive/MyDrive/telegram/shared/小歐/typhoon-tracker/state.json
```

格式：
```json
{
  "typhoonName": "米克拉",
  "status": "active",
  "activatedAt": "2026-06-24T08:00:00+08:00",
  "lastReportedAt": "2026-06-24T09:00:00+08:00"
}
```

- `status`: `active`（追蹤中）| `stopped`（已停止）| `expired`（已過期）
- 如果檔案不存在或 status 不是 `active`，就不回報。

## 運作流程

### 1. 啟用追蹤
使用者說關鍵字觸發：
- `注意[颱風名]` / `追蹤[颱風名]` / `看著[颱風名]`

→ 寫入狀態檔，啟動追蹤，立即回報最新動態。

### 2. 自動回報
**每次使用者主動找小歐對話時**（任何話題），都要先檢查狀態檔。如果有正在追蹤的颱風，在回覆正式內容之前先附上颱風快報。

### 3. 停止追蹤
使用者說關鍵字：
- `停` / `取消追蹤` / `不用看了` / `結束追蹤`

→ 更新狀態檔為已停止。

### 4. 自動過期
如果資料來源連續 3 次都找不到該颱風的資料，自動設為 `expired` 並告知使用者。

---

## 行為規則

1. **不要主動發訊息** — 只在使用者找小歐時附帶回報
2. **回報要簡短** — 3-5 行內搞定，不要喧賓奪主
3. **資料過舊判斷** — 如果上次回報在 6 小時內且使用者只是普通聊天，可以說「跟早上一樣，沒什麼變化」
4. **只用這兩個網頁** — 絕對不查 Wikipedia、JMA、JTWC、Google 或其他來源
5. **如果 Playwright 失敗** — 誠實說「氣象署網站目前無法存取，稍後再試」
